LearningConfiguration

case class LearningConfiguration(epsilon: Decay[Double] = new ExponentialDecay(0.9, 0.1, 0.01), gamma: Double = 0.9, learningRate: Double = 0.0005, batchSize: Int = 32, updateEach: Int = 100, random: Random = new Random(1), dqnFactory: DQNAbstractFactory[_], snapshotPath: String) extends Product with Serializable

The wrapper of all the hyper-parameters that the user can set

epsilon

the decay used to solve the exploitation-exploration problem

gamma

the discount factor

learningRate

the learning rate used to train the neural network

batchSize

the size of the batch of agents experience used to update the neural network

updateEach

represents every how many iterations the actual and the target neural networks are synchronized

dqnFactory

the factory of the neural network used to approximate the policy

snapshotPath

the path to save the policy snapshots
